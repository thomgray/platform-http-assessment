# platfor-http-assessment

This is the source code for the bbc-platform coding challenge

This project is built in Java 10 using maven, so ensure both are installed before building.

## Usage

### build

```sh
mvn clean install
```
or
```sh
./build
```

### test
```sh
mvn test
```
or
```sh
./test
```

### run
```sh
./run arg1 arg2 ...
```
